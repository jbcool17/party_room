# Party Roooooooooooom!

Bring the party to your browser with THREE.js!
An experimental sandbox, learning THREE.js.

### Version
1.0.0

### Tech

Party Room uses:

* React
* Rails
* jQuery
* OimoJS
* THREEx

### Installation

Clone the repo
```sh
$ git clone git@github.com:jbcool17/party_room.git
```

Install Gems
```sh
$ bundle install
```

Create a DB
```sh
$ rake db:create
```

Start up the server!
```sh
$ rails server
```
